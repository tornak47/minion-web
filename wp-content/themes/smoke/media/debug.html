Example: Javascript Communication

To watch this video, you need the latest Flash-Player and active javascript in your browser.

Javascript Communication

You can use the following methods to communicate with the player via javascript:

  • Play
  • Pause
  • Set Volume to 0
  • Set Volume to 100
  • Toggle Sound
  • Switch Color to RED
  • Reset Player
  • Seek to Second 10
  • Seek to 70%
  • The following function is sent to nonverblaster, which then calls the javascript function "recievePlayingProgress(value)": Get Playing Progress
  • The following function is sent to nonverblaster, which then calls the javascript function "recieveLoadingProgress(value)": Get Loading Progress

See the souce-code of this site if you want to know how the movie was embedded.

Important Note:

If you want to test this example offline on your computer, you have to go to the Flash Player Security Settings and give permission to the folder where these files are located on your machine.
